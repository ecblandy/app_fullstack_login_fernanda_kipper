# Projeto Fullstack - [Login page]

Este é um projeto **Fullstack** desenvolvido enquanto acompanho as aulas da **Fernanda Kipper**. O objetivo é praticar o desenvolvimento web completo, implementando funcionalidades em frontend e backend ao mesmo tempo em que aplico o que aprendo na aula.

---

## Funcionalidades

- Autenticação de usuários
- Comunicação entre frontend e backend
- Integração com banco de dados
- Deploy em ambiente de desenvolvimento

---

## Tecnologias Utilizadas

**Frontend:**

- Angular
- TypeScript
- Tailwind CSS

## Como Rodar o Projeto

### Pré-requisitos

- Node.js >= 18
- npm ou yarn

### Instalação

1. Clone o repositório:

```bash
git clone https://github.com/ecblandy/app_fullstack_login_fernanda_kipper.git
```
